# Projet 7 : Créez un réseau social d'entreprise

## Dans le **frontend**

### Setup du projet
```
npm install
```

### Compiler pour le développement
```
npm run serve
```


## Dans le **backend**

### Setup du projet
```
npm install
```

### Pour lancer le serveur de développement
```
nodemon serve
```
